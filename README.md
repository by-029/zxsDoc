您好，欢迎您使用走小散项目文档系统

本系统是一个基于 PHP 的现代化文档管理系统，支持多项目、多层级章节管理，内置 Markdown 编辑器，提供完善的搜索功能和响应式设计。

问题反馈微信公众号:走小散


功能特性

📚 文档管理

多项目支持：可以创建和管理多个文档项目

层级章节：支持无限层级的章节结构

Markdown 编辑：使用 Editor.md 编辑器，支持实时预览

伪静态 URL：友好的 URL 结构（/project-slug/chapter-slug）

🔍 搜索功能

全站搜索：在首页搜索所有项目的文档内容

项目内搜索：在项目内搜索当前项目的章节和内容

内容搜索：不仅搜索标题和路径，还搜索章节正文内容

🎨 用户界面

响应式设计：完美适配桌面端和移动端

暗色主题：支持亮色/暗色主题切换

目录导航：自动生成文档目录（TOC），支持快速跳转

章节导航：提供上一章/下一章导航按钮

美观的 UI：现代化的界面设计，优雅的滚动条样式

🔧 管理后台

项目管理：创建、编辑、删除项目

章节管理：可视化的章节树形结构管理

系统设置：配置网站基本信息

导航菜单：自定义导航菜单设置

系统要求

PHP >= 8.0

MySQL >= 5.7

Nginx 1.24.0
支持 PDO MySQL 扩展
